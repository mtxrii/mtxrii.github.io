SNAKE

 

A simple Java applet mimicking the original snake Atari game. Has an option for wrap around, where instead of ending the game when the snake hits a wall, the snake continues back on the screen from the other side.

CLOCK

 

A 7-segment display made using changing color text on a JFrame. The main class converts the current time every second into 4 bit binary arrays, and the implemented methods decode them onto the mock LED display.
